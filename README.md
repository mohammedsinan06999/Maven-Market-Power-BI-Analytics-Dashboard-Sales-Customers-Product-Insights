This project presents an interactive analytics dashboard built in Power BI using the Maven Market dataset. The dashboard provides insights into sales performance, customer demographics, product trends, and regional revenue distribution to support business decision-making.

---

## Dashboard Modules

- Top Customer Dashboard
  - Identifies top-spending customers
  - Revenue and transaction contribution analysis
  - Age-group segmentation
  - Gender distribution analysis

- Sales Performance Dashboard
  - Revenue trend over time
  - Total profit, revenue, and quantity sold
  - Monthly transaction comparison
  - Return rate and profit margin KPIs
  - Revenue by region breakdown

- Product Insights Dashboard
  - Top-selling products and brands
  - Average price and retail price metrics
  - Total products sold analysis
  - Product revenue contribution by category

- Sales by Location Map
  - Country-level filtering
  - Geo-based sales visualization
  - Regional distribution comparison
